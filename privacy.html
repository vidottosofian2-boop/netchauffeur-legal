Politique de Confidentialité

Dernière mise à jour : 22 février 2026

En bref : NetChauffeur stocke toutes vos données localement sur votre appareil. Nous ne collectons, ne transmettons ni ne vendons aucune de vos données personnelles à des tiers.

1. Responsable du Traitement

NetChauffeur est une application développée par S.V BUSINESS SERVICES.
Contact : vidotto.sofian2@gmail.com

2. Données Collectées

NetChauffeur collecte et stocke uniquement sur votre appareil les données suivantes :

  • Informations professionnelles : Courses effectuées, revenus, dépenses, heures travaillées
  • Données de localisation GPS : Trajets enregistrés pour calculer votre rentabilité kilométrique
  • Informations d'agenda : Planning de vos journées de travail
  • Données d'authentification : Si vous utilisez Google Sign-In ou Apple Sign-In (OAuth uniquement)

3. Stockage Local des Données

Toutes vos données professionnelles sont stockées localement sur votre appareil via SharedPreferences/UserDefaults. Aucune synchronisation cloud n'est activée par défaut. Vos données ne quittent jamais votre téléphone sauf si vous :

  • Générez volontairement un PDF et le partagez
  • Activez une future fonctionnalité de synchronisation (optionnelle)

4. Utilisation de la Localisation GPS

4.1 Pourquoi nous utilisons le GPS

NetChauffeur utilise votre position GPS uniquement pendant vos journées de travail pour :

  • Calculer automatiquement votre kilométrage professionnel
  • Déterminer votre rentabilité réelle au kilomètre
  • Générer des rapports de trajets pour vos déclarations fiscales

4.2 Tracking en arrière-plan

L'application peut enregistrer votre position en arrière-plan lorsque vous démarrez une journée de tracking. Vous contrôlez cette fonctionnalité et pouvez l'arrêter à tout moment depuis le dashboard.

4.3 Conservation des données GPS

Les données GPS sont stockées localement sur votre appareil et ne sont jamais transmises à nos serveurs ou à des tiers. Vous pouvez supprimer vos trajets à tout moment dans l'application.

5. Services Tiers

5.1 Firebase (Google)

NetChauffeur utilise Firebase pour :

  • Firebase Authentication : Connexion via Google/Apple (aucune donnée professionnelle transmise)
  • Firebase Analytics : Statistiques d'utilisation anonymes (nombre de courses, sessions actives)

Politique de confidentialité Firebase

5.2 Google AdMob

L'application affiche des publicités via Google AdMob. AdMob peut collecter :

  • Identifiant publicitaire (IDFA/GAID)
  • Adresse IP
  • Données de navigation dans l'app

Vous pouvez désactiver la publicité personnalisée dans les paramètres de votre appareil (iOS : Réglages > Confidentialité > Suivi / Android : Paramètres > Google > Annonces).

Politique de confidentialité Google

5.3 Google Sign-In / Apple Sign-In

Si vous choisissez de vous connecter via Google ou Apple, nous recevons uniquement :

  • Votre nom
  • Votre adresse e-mail
  • Photo de profil (optionnel)

Ces informations servent uniquement à vous identifier dans l'application. Aucune donnée n'est partagée avec ces services.

6. Achats In-App (Abonnements)

Les abonnements Premium/Pro sont gérés par Apple App Store et Google Play Store. Nous ne stockons pas vos informations de paiement. Consultez les politiques de confidentialité respectives :

  • Apple Privacy Policy
  • Google Play Privacy Policy

7. Partage de Données

Nous ne vendons ni ne louons jamais vos données. Vos informations ne sont partagées que dans les cas suivants :

  • Obligation légale : Si requis par la loi (tribunal, autorités fiscales)
  • Services essentiels : Firebase, AdMob (voir section 5)
  • Avec votre consentement explicite : Export PDF que vous partagez volontairement

8. Vos Droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès : Consulter toutes vos données dans l'application
  • Droit de rectification : Modifier vos informations depuis les paramètres
  • Droit à l'effacement : Supprimer votre compte et toutes vos données
  • Droit à la portabilité : Exporter vos données via la fonction PDF
  • Droit d'opposition : Refuser le tracking publicitaire (paramètres appareil)

Pour exercer ces droits, contactez-nous à vidotto.sofian2@gmail.com

9. Sécurité des Données

Vos données sont protégées par :

  • Stockage local chiffré : iOS Keychain / Android EncryptedSharedPreferences
  • Authentification sécurisée : OAuth 2.0 (Google/Apple)
  • Aucune transmission réseau : Données professionnelles jamais envoyées en ligne

10. Conservation des Données

Vos données sont conservées :

  • Aussi longtemps que vous utilisez l'application
  • Jusqu'à suppression manuelle de votre compte
  • Suppression automatique après désinstallation de l'app (données locales effacées)

11. Données des Mineurs

NetChauffeur est destinée aux professionnels du transport. Nous ne collectons pas sciemment de données d'utilisateurs de moins de 16 ans.

12. Modifications de cette Politique

Nous pouvons mettre à jour cette politique de confidentialité. Vous serez informé via l'application des changements importants. Date de dernière modification affichée en haut de cette page.

13. Nous Contacter

Pour toute question concernant cette politique de confidentialité :

  • Email : vidotto.sofian2@gmail.com
  • Via l'application : Menu > Support > Contact
